### 2. Описание проекта
Проект представляет собой серверное веб-приложение на базе Spring Boot для автоматизации экспертной оценки, фильтрации и ранжирования городских достопримечательностей по геолокации и пользовательскому рейтингу. 

Реализован расчет расстояний между пользователем и объектами по формуле гаверсинусов, динамическая фильтрация по категориям и минимальному порогу оценок с использованием Spring Data JPA и СУБД PostgreSQL. Интегрирована система миграций Liquibase и реализовано контейнеризированное развертывание через Docker.

### 3. Подготовительные действия (установки, настройки и т.д.) для успешной работы проекта
1. Установить JDK 21 и Apache Maven.
2. Установить и запустить СУБД PostgreSQL (создать пустую базу данных с именем `city_guide`).
3. Проверить настройки подключения к БД в конфигурационном файле приложения (`src/main/resources/application.properties`):
   * `spring.datasource.url=jdbc:postgresql://localhost:5432/city_guide`
   * `spring.datasource.username=postgres`
   * `spring.datasource.password=your_password`

### 4. Информация о доступах 
* **URL подключения к БД**: `jdbc:postgresql://localhost:5432/city_guide`
* **Базовый URL для REST API запросов**: `http://localhost:8080/api`

### 5. Описание, как запустить ваш проект
1. Склонировать репозиторий с решением
Перейти в корневую директорию проекта, где находится файл pom.xml.
Выполнить сборку проекта и автоматический запуск миграций Liquibase через Maven:mvn clean install
Запустить Spring Boot приложение:mvn spring-boot:run
Приложение начнет принимать HTTP-запросы на порту 8080.
